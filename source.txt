Columns++ is a plugin for Notepad++ offering various features
for working with text and data arranged in columns.

Source code repository: https://github.com/Coises/ColumnsPlusPlus

Columns++ for Notepad++ was first released by Randall Joseph Fellmy
(<software@coises.com>, <http://www.coises.com/software/>) in 2023.
